# Rule-Based Chatbot using Python

## Description
This project is a simple rule-based chatbot built using Python. It responds to user inputs using predefined rules with if-elif statements.

## Features
- Time-based greetings
- Answers basic questions
- Shows current time and date
- Tells jokes
- Responds to user inputs like mood, favorite color, thanks
- Exit using 'bye'

## Technologies Used
- Python
- Random module
- Datetime module

## How to Run
1. Open terminal in project folder
2. Run: python chatbot.py
3. Start chatting

## Sample Inputs
hi, hello, how are you, time, date, joke, mood, favorite color, thanks, bye
